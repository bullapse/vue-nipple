# vue-nipple
Vuejs component wrapper for [yoannmoinet's nipplejs](https://github.com/yoannmoinet/nipplejs)

# Installation
## Vue Support
Supports Vue >= 2

## Installation and Usage
`npm install vue-nipple --save`

```
import Nipple from 'vue-nipple';

Vue.component('vue-nipple', Nipple)
```
